____________________________________________SETUP_________________________________________________________________________________________				

1) Start the Apache and MySQL modules using the WAMP controller.
2) Open the phpMyAdmin and create a database named 'ctrl_budget'. 
3) Import the ctrl_budget.sql file present in the zip folder.
4) Open the www folder in the wamp64 folder. Copy paste the folder myproject.
5) Open the browser (say, Google Chrome) and then type: localhost/myproject/index.php and the expense manager website named Ct₹l budget shall be opened..
